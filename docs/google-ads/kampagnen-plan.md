## Optimierungstipps

1. **Erste 2 Wochen:** Daten sammeln, keine großen Änderungen
2. **Nach 2 Wochen:** Schwache Keywords pausieren (CTR < 2%)
3. **Nach 4 Wochen:** Gebotsanpassungen basierend auf Conversion-Daten
4. **Monatlich:** Suchanfrageberichte prüfen, neue negative Keywords hinzufügen
5. **Saisonal:** Budget im Winter erhöhen (mehr Verstopfungen durch Frost)

## Import in Google Ads Editor

Die CSV-Datei `google-ads-import.csv` kann direkt in Google Ads Editor importiert werden:
1. Google Ads Editor öffnen
2. Konto auswählen
3. "Import" → "Aus Datei importieren"
4. CSV-Datei auswählen
5. Änderungen überprüfen
6. "Posten" klicken
