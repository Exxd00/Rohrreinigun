# Rohrreinigung Kraft

Professionelle Rohrreinigung & Kanalreinigung in Mittelfranken.

**Website:** [rohrreinigung-kraft.de](https://rohrreinigung-kraft.de)

## Über das Projekt

Rohrreinigung Kraft ist ein lokaler Fachbetrieb für Rohrreinigung und Kanalreinigung in Mittelfranken. Das Servicegebiet umfasst Nürnberg und einen Umkreis von 60 km.

### Hauptstädte

- **Nürnberg** - Hauptsitz
- **Fürth** - Direkter Nachbar
- **Erlangen** - Universitätsstadt

### Servicegebiet

60 km Umkreis von Nürnberg, inkl. Schwabach, Roth, Ansbach, Forchheim, Neumarkt i.d.OPf. und viele weitere Städte.

## Technologie

- **Framework:** Next.js 15 (Turbopack)
- **Styling:** Tailwind CSS + shadcn/ui
- **Sprache:** TypeScript
- **Hosting:** Vercel

## Entwicklung

```bash
# Installation
bun install

# Entwicklungsserver starten
bun run dev

# Build erstellen
bun run build
```
