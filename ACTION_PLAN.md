## 🔴 الإجراءات الفورية (هذا الأسبوع)

### 1. حل مشكلة السعر الفوري

**المشكلة:** المنافسون يعلنون "ab 19€" وأنت "ab 79€"

**الحل في 3 خطوات:**

#### خطوة 1: تغيير رسالة السعر
```
❌ القديم: "Rohrreinigung ab 79€"

✅ الجديد: "Rohrreinigung ab 79€ - FESTPREIS vor Arbeit"
            "Andere bieten 19€ - aber was bekommen Sie wirklich?"
            "Bei uns: Preis VORHER = Preis NACHHER. Garantiert."
```

#### خطوة 2: إنشاء صفحة مقارنة
**URL: /preisvergleich أو /warum-mehr**

محتوى الصفحة:
```
Warum kostet Rohrreinigung Kraft mehr?
--------------------------------------
Weil Sie bei uns MEHR bekommen:

❌ Billig-Anbieter (ab 19€):
- Preis steigt vor Ort
- Keine Garantie
- "Das kostet extra..."
- Anfahrt 50€+

✅ Rohrreinigung Kraft (ab 79€ Festpreis):
- Preis VORHER genannt = Endpreis
- 7-90 Tage Garantie
- Diagnose kostenlos
- Anfahrt inklusive
- Nein gesagt? 0€ gezahlt

Rechnung:
Billig-Anbieter: 19€ + 50€ Anfahrt + 60€ "Spezialgerät" + 40€ "schwere Verstopfung" = 169€
Kraft: 79€ Festpreis = 79€

WER IST WIRKLICH BILLIGER?
```

#### خطوة 3: إعلان مواجه للمنافسين
```
Headline: "19€ Rohrreinigung? Fragen Sie nach dem ENDPREIS!"
Description: "Festpreis 79€ = wirklich 79€. Keine Überraschungen. Keine Extras. Garantiert."
```

## 🟡 الإجراءات هذا الشهر

### 5. Wartungsvertrag = دخل شهري ثابت

**هذا أهم شيء للمال!**

**إنشاء 3 باقات:**

#### Basis-Wartung (für Hausbesitzer)
```
29€/Monat
- 2x jährliche Inspektion
- 1x jährliche Reinigung
- 20% Rabatt auf Notdienst
- Priorität bei Notfall
```

#### Komfort-Wartung (für Mehrfamilienhaus)
```
49€/Monat pro Einheit
- 4x jährliche Inspektion
- 2x jährliche Reinigung
- Alle Abflüsse
- Dokumentation
- Notdienst in 30 Min
```

#### Business-Wartung (für Hausverwaltung/Gewerbe)
```
Individuell
- Rahmenvertrag
- Sammelrechnung
- Fester Ansprechpartner
- 24/7 Priorität
```

**الهدف:**
- شهر 1: 10 عقود = 290€/Monat
- شهر 3: 30 عقود = 870€/Monat
- سنة 1: 100 عقود = 2,900€/Monat = 34,800€/Jahr

### 6. Upselling Script

**بعد كل Soforthilfe (79€):**

```
Techniker: "Problem ist gelöst! Aber ich sehe [Fett/Haare/Ablagerungen].
            Das kommt in 3-6 Monaten wieder.

Option 1: Ursachen-Paket heute: +200€
          - Kamera-Inspektion
          - Ursache beheben
          - 90-Tage-Garantie

Option 2: Wartungsvertrag: 29€/Monat
          - Problem kommt nie wieder
          - Regelmäßige Reinigung
          - 20% Rabatt immer

Was passt Ihnen besser?"
```

**معدل التحويل المتوقع:** 20-30%
**ربح إضافي:** 50€-100€ لكل زيارة
